HOMEWORK 3 - due Thursday, October 30th no later than 5:00PM

Write a C or C++ program that displays a text file on the screen 20 lines at a time. The input file name should be given as a command line argument. The program should display the next 20 lines after N has been typed, or display previous 20 lines after P has been entered (this is an elementary version of the more utility).

In Windows, the command to clear the screen is cls; in UNIX, it is clear. Try one of these commands on your system so that you understand its effects. Then use either system("cls") or system("clear") in your program before you write each set of lines to the screen.

The input is case-insensitive, i.e. "n" and "N" or "p" and "P" are the same.

GRADING KEY

  • (5 pts): Proper use of Comments.
    Include your name, assignment number and a brief description of the program at the top of main program.
  • (5 pts): Good use of Functions.
  • (5 pts): Inclusion of a makefile.
  • (5 pts): Appropriate #define statements are included.
  • (10 pts): Error message displayed when a problem reading information from a file.
  • (30 pts): Program compiles without error.
  • (20 pts): Program displays the next set of lines.
  • (20 pts): Program displays the previous set of lines.
  • EXTRA CREDIT (Optional):
    • (7 pts): Allow a command line option of the form -n to be used, where n is a positive integer specifying the number of lines that are to be displayed at one time.
    • (13 pts): Display more than one file given as command line arguments.
